Savez-vous jouer à Prime Palindome Golf ? Il s'agit, à partir d'un nombre entier donné, de trouver l'entier supérieur le plus proche qui soit un nombre premier et palindrome.

Un nombre palindrome est un nombre dont l'écriture décimale se lit de la même façon de la droite vers la gauche et de la gauche vers la droite. 79197 en est un exemple. L'écriture de ces nombres est symétrique.
Un nombre premier est un nombre entier naturel supérieur à 2 qui n'a pas d'autre diviseur positif que 1 et lui-même.
Dans ce défi, un nombre entier vous est donné. Il faut trouver l'entier le plus proche qui:
- est strictement plus grand que le nombre donné ;
- est premier ;
- est palindrome.
Par exemple : pour le nombre 13, le plus proche premier palindrome supérieur est 101. Pour le nombre 2, la réponse est 3 car un chiffre est palindrome.

Il y a une règle supplémentaire pour ce défi. C'est une mission de code golf et l'objectif est de faire le code le plus court possible. Plus le code est court, plus vous gagnez de points. Le score pour cette mission est dynamique et calculé en fonction de la longueur du code. Votre code doit faire moins de 250 caractères. Pour chaque caractère de moins que 250, vous gagnez 1 point. Par exemple, un code de 200 caractères rapporte 50 points.

Important: Nous utilisons la limite de profondeur de récursivité par défaut (100). Alors n'essayez pas de résoudre récursivement cette mission.

Input: Un nombre entier naturel (int).

Output: Le nombre premier palindrome supérieur le plus proche (int).

Exemple:

golf(2) == 3
golf(13) == 101
golf(101) == 131

Utilisation: Les nombres premiers sont très utilisés dans de nombreux domaines. Ce défi ne traite pas d'élégance ou de performance de code. Il s'agit de trucs et astuce de Python pour réduire la longueur le code. Cela peut aider à une meilleure compréhension de Python.

Précondition: 0 < número < 98689
